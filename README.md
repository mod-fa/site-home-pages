# site-home-pages

这是一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要存放各类静态 HTML 页面文件，内容涵盖不同主题与用途。每个页面均为独立完整的 HTML 文档，不依赖特定后端服务或数据库。页面以纯前端方式构建，可直接在浏览器中打开使用。

## 目录说明

```
/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（图片、样式、脚本等）
├── docs/           # 相关文档或说明文件
└── README.md       # 本文件
```

- `pages/` 目录下每个子文件夹代表一个独立页面项目，文件夹内包含对应的 `index.html` 及配套资源。
- `assets/` 目录存放可复用的公共资源，如通用样式表、图标、字体文件等。
- `docs/` 目录可选，用于放置页面使用说明、设计思路或变更记录。

## 页面归档说明

- 每个 HTML 页面均为自包含文件，原则上不依赖外部链接（除 CDN 公共库外）。
- 页面文件名应清晰反映其内容或功能，避免使用无意义命名。
- 归档时建议在 `pages/` 下新建独立文件夹，避免文件混乱。
- 若页面需引用本地资源，请统一放置在 `assets/` 对应子目录下。

## 维护说明

- 本仓库欢迎提交新页面归档，但请确保内容合法、不侵犯他人权益。
- 提交前请检查 HTML 代码是否规范，资源引用路径是否正确。
- 若需更新已有页面，请直接在对应文件夹内修改，并在提交信息中注明变更内容。
- 不建议在此仓库中存放动态交互复杂或依赖私有 API 的页面。
- 仓库维护者保留对提交内容进行整理、归类或移除的权利。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 在 `pages/` 目录下找到所需页面文件夹。
3. 直接双击打开 `index.html` 即可在浏览器中查看。
4. 也可将整个仓库部署到任意静态托管服务，通过 URL 访问各页面。
